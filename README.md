Задача.
Необходимо разобраться в данных, поступивших с веб-сайта, провести базовую чистку данных (дубликаты, пустые значения, типизация данных, ненужные атрибуты).
Создать модель предсказания совершения целевого деййствия (значение roc_auc > 0,65) - факт совершения пользователя целевого действия.
Упаковать получившуюся модель в сервис, который на вход берет определенные атрибуты и отдает на выходе 0/1 (1 - если пользователь всовершит любое целевое действие).
